Advanced Settings

DeepTeach uses a bagged boosted decision tree as its supervised classifier. The features it uses are the representation learned by a deep convlutional neural network. The options below give control over some of the BBDT's hyper-parameters:

Percentage of random features

Controls the number of features to randomly use when deciding on the best split. This controls the random_feature_propn configuration parameter. See the Classifier configuration documentation page for more details. It can be used to limit the model's learning capacity and prevent overfitting.

Numbers of bags

Controls the number of bags to use when training. Each bag is a random selection of training examples on which one tree will be trained. Bagging helps with generalization, but slows down training. The higher the number, the longer the training will take.
